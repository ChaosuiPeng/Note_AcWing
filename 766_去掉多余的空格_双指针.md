### 766.去掉多余的空格
输入一个字符串，字符串中可能包含多个连续的空格，请将多余的空格去掉，只留下一个空格。

**输入格式**

共一行，包含一个字符串。

**输出格式**

输出满足要求的矩阵。

**数据范围**

输入字符串的长度不超过 200。

保证输入字符串的开头和结尾没有空格。

**输入样例：** 

Hello      world.This is    c language.

**输出样例：** 

Hello world.This is c language.

### 思路

A) 直接用cin过滤空格
B) 双指针

### 代码
```c++
#include <iostream>
using namespace std;
int main()
{
    string s;
    while(cin >> s) cout << s << " ";
    return 0;
}

```
